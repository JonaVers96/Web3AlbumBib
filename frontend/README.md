# Album Store

Een full-stack e-commerce applicatie voor de verkoop van muziekalbums, gebouwd als eindproject.

## Features
* **Volledige webshop:** Albums bekijken, zoeken en filteren op artiest.
* **Winkelmand & Checkout:** Albums toevoegen aan winkelmand en veilig afrekenen via de Mollie API.
* **Admin Dashboard:** Beheer van albums (inclusief image uploads), artiesten en gebruikers.
* **Authenticatie:** Veilige login en registratie met JWT-tokens en role-based access (admin/user).

## Tech Stack
* **Frontend:** React 19, TypeScript, Tailwind CSS v4, React Router v7, React Hook Form.
* **Backend:** Node.js, Prisma ORM, MySQL (Aiven).
