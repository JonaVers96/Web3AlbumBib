# Webify Starterproject

## Over dit project

Dit is het startersproject voor de Webify muziekapplicatie. In deze applicatie werk je met React, TypeScript en Tailwind CSS. De basisstructuur, styling en voorbeelddata zijn al voorzien. Het is de bedoeling dat jij de ontbrekende functionaliteit toevoegt door de TODO’s in de code te volgen.

## Wat wordt er verwacht?

- **Volg de TODO’s:** In de codebestanden vind je overal duidelijke TODO-commentaren. Deze geven aan waar je zelf functionaliteit moet toevoegen of code moet aanvullen.
- **Implementeer de logica:** Je zorgt ervoor dat albums en nummers correct getoond worden, dat je tussen pagina’s kan navigeren, en dat de afspeelfunctionaliteit werkt.
- **Gebruik context en componenten:** Je maakt gebruik van React context voor het beheren van de afspeelstatus en de playlist. Je werkt met herbruikbare componenten.
- **Styling is voorzien:** Je hoeft zelf geen extra styling toe te voegen. Focus op de werking en structuur van de applicatie.
- **Werk modulair:** Houd je code overzichtelijk door logica en componenten te scheiden maar je moet geen nieuwe componenten aanmaken.
